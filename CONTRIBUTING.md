# Contributing

## Branch Naming
- feat/*
- fix/*
- chore/*
- docs/*
- refactor/*

## Commit Convention
Use conventional commits:

- feat(scope): message
- fix(scope): message
- docs(scope): message
- chore(scope): message
- refactor(scope): message
- test(scope): message
- ci(scope): message

## Rules
- No direct push to main
- Use pull requests
- Keep documentation updated
- Follow project architecture and coding standards
