# RoomGenius

RoomGenius is a Flutter mobile app that helps users create professional AI-ready interior design prompts from room photos and design preferences.

## Product Positioning

This MVP does not generate AI images inside the app.  
Instead, it helps users build structured prompts that can be used with tools like ChatGPT, DALL·E, Midjourney, Leonardo, or Stable Diffusion.

## Core Features

- Room project management
- Room type and style selection
- AI-ready prompt generation
- Local-first saved projects
- Before / after comparison
- Clean Architecture
- BLoC / Cubit state management
- GoRouter navigation
- GetIt dependency injection

## Tech Stack

- Flutter
- Dart
- BLoC / Cubit
- GoRouter
- GetIt
- SharedPreferences / local storage
- Firebase Core, optional
