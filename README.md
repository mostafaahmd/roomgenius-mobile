# RoomGenius Mobile

RoomGenius is a Flutter mobile application that helps users redesign room interiors from a room photo using AI-generated design concepts.

## Current Status
Sprint 0 — Foundation Lock

## Goals
- Build a production-minded Flutter architecture
- Use local-first data flow
- Integrate Supabase for backend services
- Use Gemini through a secure backend bridge later
- Keep the project clean, documented, and portfolio-ready

## Planned Core Features
- Onboarding
- Create design flow
- AI generation
- Results comparison
- Local projects
- Cloud sync
- Settings

## Tech Stack
- Flutter
- Cubit
- go_router
- get_it
- dio
- Drift
- Supabase
- Gemini
- freezed / json_serializable
